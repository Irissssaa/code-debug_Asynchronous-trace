### 项目描述

针对Rust语言的异步函数调用跟踪与调试难以及已有方法通用性差的问题，本项目设计基于GDB的白名单动态函数插桩跟踪与调试方法。我们旨在完成以下四个目标：

- 目标1：获取Rust异步代码与运行状态的对应关系；
- 目标2：获取Rust程序的Future依赖关系；
- 目标3：基于GDB实现支持白名单的异步函数跟踪框架；
- 目标4：Rust异步函数运行数据的可视化；

```
├── results/        			# 分析结果输出
├── tests/          			# 测试项目

```
